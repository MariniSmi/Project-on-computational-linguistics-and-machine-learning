Продолжение проекта "Количественный анализ лингвистических особенностей текстов разных уровней CEFR (A1–C1) на основе метрик сложности" с 1 семестра обучения.

Тема проекта: Применение машинного обучения для классификации аугментированных текстов по уровням CEFR(A1-C1) методом back translation

Объект исследования: Корпус из 1400 аугментированных текстов уровней A1, A2, B1, B2, C1. Тексты на разные тематики, разных стилей и степени сложности.

Цель проекта: опробовать обучение модели машинного обучения на базе текстов, аугментированных с помощью обратного перевода.

Задачи проекта:

1) Увеличение объема корпуса текстов
2) Кодирование текстовой информации
3) Обучение модели машинного обучения
4) Сравнение результатов


Этапы проведения анализа:
1) Увеличение объема корпуса текстов с помощью генерации текстов через API Gigachat
2) Увеличение объема корпуса текстов с помощью аугментации текстов методом обратного перевода (английский - немецкий - английский)
3) Кодирование текстовых данных двумя разными способами: TF-IDF и предобученного BERT
4) Обучение модели CatBoost
5) Сравнение результатов работы модели CatBoost с учетом использованного метода кодирования текстов.
